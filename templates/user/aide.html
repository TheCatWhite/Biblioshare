📖 Guide d’utilisation

Vous pouvez chercher des ressources via la barre de recherche ou parcourir la liste complète.

❓ FAQ


  • Comment trouver une ressource ? → Allez dans “Accueil” et rechercher votre ressources.
  • Comment modifier mon profil ? → Dans la page profil, cliquez sur “Modifier” puis soumettre.
  • Comment changer mon mot de passe ? → Dans la section profil, onglet “Changer mot de passe”.
  • Comment uploader une ressource ? → Allez dans “Profil” et remplissez le formulaire.
  • Comment voir les ressources que j' ai uploader ? → Dans la page profil, cliquez sur “voir mes ressources partagées”.
  • Comment voir la liste de toutes les ressources partagées? → Dans la section "Ressource", tu peux alors voir la lites des ressources et des propriétaires.
