Guide d’utilisation

Bienvenue sur Biblioshare ! Voici comment tirer le meilleur parti de notre plateforme :

  • 🔍 Rechercher une ressource : Utilisez la barre de recherche sur la page d’accueil pour trouver des ressources par mot-clé, type ou propriétaire.
  • 📂 Parcourir toutes les ressources : Consultez la liste complète dans la section "Ressources".
  • 📝 Uploader une ressource : Depuis votre profil, cliquez sur "Uploader une ressource" et remplissez le formulaire.
  • 👀 Voir vos ressources : Cliquez sur "Voir mes ressources partagées" dans votre profil.
  • ⚙️ Modifier votre profil : Dans la page profil, cliquez sur "Modifier le profil" et enregistrez vos changements.
  • 🔑 Changer votre mot de passe : Depuis votre profil, sélectionnez "Changer le mot de passe".

FAQ

Allez sur la page d’accueil et utilisez la barre de recherche pour entrer le mot-clé ou le nom du propriétaire.

Dans votre page profil, cliquez sur "Modifier le profil", faites vos modifications et enregistrez.

Depuis votre profil, cliquez sur "Uploader une ressource" et remplissez le formulaire avec le fichier et le type de ressource.
